Our process:

Initial consultation is done to gather likes, dislikes, target audience, essential needs for the project. Contract outlining project scope, deliverable timeline, and payments is agreed to and signed by both parties.

A wireframe mockup/prototype of the initial front page of the web site is created for the client for inspection to ensure all items meet the client's needs and wants.

Once the prototype is approved, the actual programming and site development begins. The client is given a test domain on our web site to view the project as it is being developed.

Upon completion of the project and the client is fully satisfied with the web site, all source-code and graphics are then delivered via a DVD-ROM as well as a fully published to the internet for live operation.
