Our Website Creation Process:

1. DISCUSS

Initial consultation is done to gather likes, dislikes, target audience, essential needs for the project. Contract outlining project scope, deliverable timeline, and snippets is agreed to and signed by both parties.

2. DESIGN

A wireframe mockup/prototype of the initial front page of the web site is created for the client for inspection to ensure all items meet the client's needs and wants.

3. DEVELOP

Once the prototype is approved, the actual programming and site development begins. The client is given a test domain to view the project as it is being developed.

4. DEPLOY

Upon completion of the project and the client is fully satisfied with the web site, all source-code and graphics are then delivered via a DVD-ROM as well as a fully published to the internet for live operation.
